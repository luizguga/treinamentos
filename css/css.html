CSS (Cascading Style Sheets)

CSS inline

É a técnica mais básica para aplicar estilos em áreas pontuais em nosso site, que é usando as CSS dentro de parâmetros de HTML5.

<body style="font-family: Arial, Helvetica, sans-serif; color: blue;">
 <h1 style="color: red;">Capítulo 1</h1>

CSS internal style

Para aplicar estilos de forma mais dinâmica e prática, podemos adicionar uma tag <style> dentro da área <head>

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Estilos internos</title>
  <style>
    body {
      font-family: Arial, Heveltica, sans-serif;
    }
  </style>
</head>

CSS external style

Manter as folhas de estilo fora do código HTML, além de uma maior organização faz com que tudo seja reaproveiado de maneira mais eficiente nas outras páginas do nosso site. Para isso, utilizamos a tag <link> especialmente configurada para trabalhar com arquivos externos de estilo. Essa tag deve ser colocada dentro da área <head> do seu documento HTML.

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS external</title>
  <link rel="stylsheet" href="style.css">
</head>

Agora é só adicionar os seletores e todas as suas respectivas declarações nesse arquivo separado.
