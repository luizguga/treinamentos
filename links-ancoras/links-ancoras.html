Links e Âncoras em HTML5

Tipos de links

Link Interno

Leva o visitante a outra página dentro do próprio site. Não é necessário indicar a URL completa nesses casos.

link interno (para a próxima página do site) link interno (para a página anterior do site)

Link Externo

Devemos indicar a URL completa, incluindo o protocolo http:// ou https:// e o caminho que leve à uma página específica.

link externo (site do autor de um artigo) link externo (para outro site) link externo (para um site não endossado)

Links para Download

Basta fazer o link diretamente para o arquivo que se deseja efetuar o download e adicionar o atributo download com o valor configurado para o nome do arquivo a ser baixado e o atributo type para indicar ao navegador que tipo de arquivo está sendo baixado.

link para download
